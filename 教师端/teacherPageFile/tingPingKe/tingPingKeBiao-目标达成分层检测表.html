1.课堂常规

1.1 运动参与达成

学生乐于学习，主动参与，为本课学习与锻炼做好铺垫

1.2 运动技能

为学习体育运动知识，掌握运动技能，增强安全意识和防范能力做好铺垫。

1.3 身体健康达成

学生是否掌握健康知识与方法；是否为发展学生体能与健身能力做好铺垫。

1.4 心理健康与社会适应达成

学生是否在活动中表现出合作意识和良好的体育道德，能为完成学习任务做好铺垫。

2. 准备部分

2.1 运动参与达成

学生乐于学习，主动参与，取得不同进步，为本课学习和锻炼做好准备。

2.2 运动技能

增强安全意识和防范能力，为学习体育运动知识，掌握运动技能做好准备。

2.3 身体健康达成

是否发展学生体能与健身能力；为掌握健康知识与方法做好准备。

2.4 心理健康与社会适应达成

学生是否在活动中表现出合作意识和良好的体育道德，能为完成学习任务做好准备。

3. 基本部分

3.1 运动参与达成

学生乐于学习，主动参与，取得不同进步。

3.2 运动技能

学习体育运动知识，掌握运动技能与方法，增强安全意识和防范能力。

3.3 身体健康达成

学生是否掌握健康知识与方法；是否发展学生体能与健身能力。

3.4 心理健康与社会适应达成

学生是否能够完成学习任务，活动中表现出合作意识与能力；表现出良好的体育道德和坚强的意志品质。

4. 结束部分

3.1 运动参与达成

学生乐于学习，主动参与，继续进步。

3.2 运动技能

巩固运动技能与方法，强化安全意识和防范能力。

3.3 身体健康达成

学生是否巩固健康知识与方法；学生体能与健身能力发展得到延伸。

3.4 心理健康与社会适应达成

学生是否巩固学习任务，活动中强化合作意识与能力；表现出良好的体育道德。
